# 60SecondSweep
Sixty Second Sweep is a multiplication speed game built in Javascript using the Phaser.io game engine.  It uses the OpenWS JSON Object Server to store an online scoreboard.

The Sixty Second Sweep is a great way to practice your multiplication facts. There are 31 products in the honeycomb that represent all 36 pairs of factors possible with the digits 2 â€“ 9. The numbers in the middle row each have two pairs of factors.  Type the factors for the current hexagon to move onto the next one.  Type the factors for all 31 hexagons in fewer than 60 seconds and you win!
